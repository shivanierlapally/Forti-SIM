# Forti-SIM

Forti-SIM is an innovative prototype that integrates a fingerprint sensor with an Arduino Uno microcontroller to secure SIM tray access. The system allows the SIM tray to open only when an authorized fingerprint is detected. In case of unauthorized attempts, the system triggers a security alert by sending a recovery email notification to the user. This approach ensures both physical security and real-time monitoring, offering a smart alternative to conventional SIM tray mechanisms.

Tech Stack-
Microcontroller: Arduino Uno
Fingerprint Sensor
GPS Module
LED Display
Wires and circuitry for integration
Arduino code for fingerprint authentication and alert system


How It Works

When a fingerprint is scanned, the system checks for authorization. If recognized, the SIM tray unlocks. If not, a security alert is triggered, and an email is sent immediately. This provides both physical and digital protection for mobile access.
